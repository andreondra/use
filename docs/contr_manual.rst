Contributor's Guide
##################

.. toctree::
   :maxdepth: 2
   :caption: Contents:

If you want to contribute to the USE project, there are two options how to do so.

Opening an issue
================
If you do not like anything, miss some feature, found some bug(s) or you have an idea how to improve to project,
do not worry to open the issue on GitHub.

Submitting a pull request
=========================
If you added a new system, component or any other enhancement, you may open a pull request.
