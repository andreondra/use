bool readConfirmed(uint32_t address, uint32_t &buffer)

Read data at a specified address with confirmation that anybody actually responded. Used for e.g. buses, when there does not have to be a device responding to the specified address.

Parameters

  • address – An address to read from.

  • buffer – A buffer to store the value to.

Returns

true If any Component responded to the address.
